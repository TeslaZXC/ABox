# 🌌 ABox — Твоё Облако в Стиле «Жидкого Стекла»

ABox — это не просто файлообменник, это целая экосистема персонального облачного хранения с премиальным дизайном **Glassmorphism**. Проект объединяет современный фронтенд на Vue 3 и мощный бэкенд на FastAPI с поддержкой **S3-совместимого протокола**.

---

## 🎨 Дизайн и Философия: Liquid Glass
Центральная идея ABox — превратить работу с файлами в эстетическое удовольствие. 
- **Эффект стекла**: Всё приложение построено на глубоких размытиях (`backdrop-filter: blur`), полупрозрачных градиентах и мягких тенях.
- **Интерактивность**: Микро-анимации при наведении на файлы, плавные переходы между разделами и «живые» карточки, которые реагируют на действия пользователя.
- **Адаптивность**: Интерфейс одинаково роскошно смотрится как на десктопе, так и на мобильных устройствах.

---

## 🏗 Как всё устроено (Архитектура)

Проект разделен на две основные части, которые общаются между собой через высокопроизводительное API.

### 1. Бэкенд: Мозг на FastAPI
Бэкенд выполняет две роли одновременно: стандартное REST API для фронтенда и **S3-шлюз**.

- **FastAPI**: Выбран за счет высокой скорости и автоматической генерации документации.
- **SQLAlchemy + SQLite**: Используются для хранения метаданных. В базе (`abox.db`) живут пользователи, их бакеты, ключи доступа и информация о каждом файле (размер, ETag, MIME-тип).
- **Физическое хранилище**: Сами файлы лежат на диске в папке `backend/storage/`. Структура папок повторяет логику S3: `{user_id}/{bucket_name}/{object_key}`.

#### 🛰 S3 Интеграция (Сердце системы)
В модуле `api/s3.py` реализована магия эмуляции AWS S3:
- **XML-ответы**: Бэкенд формирует XML-структуры, которые в точности соответствуют стандартам Amazon. Это позволяет сторонним клиентам «думать», что они работают с настоящим S3.
- **Стриминг данных**: При загрузке (`PUT`) файлы не читаются целиком в память, а передаются потоком напрямую на диск, что экономит ресурсы сервера.
- **Etag и MD5**: При каждой загрузке рассчитывается контрольная сумма файла, что гарантирует целостность данных.

### 2. Фронтенд: Красота на Vue 3
Фронтенд — это лицо ABox, созданное с использованием самых современных инструментов.

- **Vite**: Обеспечивает мгновенную перезагрузку при разработке и оптимизированную сборку.
- **Vuetify**: Используется как база компонентов, но полностью переработан с помощью кастомных CSS-переменных для достижения эффекта стекла.
- **Pinia**: Хранилище (`store`), которое управляет состоянием бакетов и файлов, обеспечивая мгновенное обновление списка при загрузке или удалении.
- **Система модальных окон**: Кастомные диалоги подтверждения удаления (вместо стандартных алертов браузера) поддерживают общий стиль.

## 🔒 Безопасность и Авторизация
- **JWT (JSON Web Tokens)**: Используются для обычной работы в браузере.
- **Access & Secret Keys**: Каждому пользователю при регистрации выдаются ключи доступа (как в настоящем облаке), которые используются для подписи запросов к S3-слою.
- **Квоты**: Система строго следит за объемом дискового пространства каждого пользователя (по умолчанию 10 ГБ).

---

## 🚀 Основные возможности
1. **Управление бакетами**: Создавайте и удаляйте изолированные хранилища под разные проекты.
2. **Умный диск**: Загружайте файлы методом Drag-and-Drop, перемещайтесь по папкам.
3. **Медиа-Вьювер**: Смотрите фото, слушайте музыку и запускайте видео прямо в облаке без скачивания.
4. **Удаление без следов**: При удалении бакета физически очищаются все связанные файлы на сервере.

## 🛠 Запуск проекта

### Бэкенд
1. Перейдите в `backend`
2. Создайте venv: `python -m venv venv`
3. Установите зависимости: `pip install -r requirements.txt` (FastAPI, SQLAlchemy, uvicorn, pydantic-settings)
4. Запустите: `python main.py`

### Фронтенд
1. Перейдите в `frontend`
2. Установите пакеты: `npm install`
3. Запустите: `npm run dev`
